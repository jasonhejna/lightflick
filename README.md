# lightflick
Trigger events by flicking your lights on and off a couple times
>```
          .----------.
          |   ~ON~   |
          |   ____   |
          |  |.--.|  |
          |  ||  ||  |
          |  ||__||  |
          |  ||\ \|  |
          |  |\ \_\  |
          |  |_\[_]  |
          |          |
          |  ~OFF~   |
          '----------'
>```

Measure a photoresistor value and trigger events if the light are flicked on/off at least twice. Number of flicks can be set.
